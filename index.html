NCR

NCR is a software and hardware company that is an enterprise technology leader for restaurants, retailers and banks. They have been handling transactions for 134 years, from building the very first cash registers to building POS applications and ATM software/hardware. NCR is the #1 global POS software provider for retail and hospitality, and the #1 provider of ATM software as well. NCR focuses on building meaningful relationships with every transaction. Thus, they help their clients bridge digital and physical operations by creating software, hardware, and services that can run their enterprise, from the backend to frontend.

JOB DESCRIPTION

I was hired to be a part of the new SmartLockers team at the NCR Waterloo office as a Software Developer for 8 months. I primarily worked on React Web applications and utilized the in-house software to build an orchestration layer that the UI can utilize to communicate with the backend services. My day-to-day activities involved working on new feature development, bug fixing, code review, and testing on the Teller and POS Web applications. This type of work exercised my attention to detail, communication and problem-solving skills as it involved me to properly design, plan, and communicate with the team to develop new features or resolve bugs.

Term Focus

I am working towards learning about Full Stack Development using JavaScript/TypeScript. This work term has helped me accomplish many of the skills that I require to become a Full Stack Developer. I have had the opportunity to work on React (frontend), API Orchestration layer and backend services and technologies. I will take the skills that I have attained from NCR and utilize them during my academic studies.

The focus of the work term report was Full Stack Development, agile methodologies and attaining tips to become a professional software developer. From my mentors at NCR, I have learned that a good software developer is not one that can finish a task the fastest, but a good software developer is one that can properly design a solution, come up with edge cases to test and properly document their work before submitting.
